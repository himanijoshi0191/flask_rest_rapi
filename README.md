# flask_rest_rapi
# flask_project_web
Flask Restful API (Create, Read, Update, Delete)

## Prerequisites:
* python 3.6+
* Virtual Environment

## virtual environment:

* python -m venv venv
## activate:
* source venv/bin/activate(linux)
* venv\Scripts\activate(windows)

## Installation and running:
* pip install -r requirements.txt
* cd flask_rest_rapi
* python dispatcher.py
